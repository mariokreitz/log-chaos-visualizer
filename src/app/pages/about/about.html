What this project does

This application visualizes and analyzes log files to surface errors, fatals, and trends over time. It provides interactive charts and tables to help explore large log datasets efficiently.

Development & architecture

The project follows a modern Angular standalone-component architecture with a focus on small, focused components and lazy-loaded routes. State is managed with Signals and derived values use `computed()` where appropriate.

Technologies used

  • Angular (standalone components, Signals)
  • TypeScript with strict typing
  • Angular Material for accessible UI components
  • Chart.js / ng2-charts for visualizations
  • Web Workers for heavy log parsing
